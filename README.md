# DotTraceIP

![Python](https://img.shields.io/badge/Python-3.12%2B-blue)
![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey)
![Category](https://img.shields.io/badge/Category-Networking-orange)

Интерактивный многопоточный инструмент для сетевого анализа и массового сбора информации об IP-адресах. Поддерживает работу через HTTP, SOCKS4 и SOCKS5 прокси.

## Установка

Клонируйте репозиторий и установите зависимости:

```bash
git clone https://github.com/network-user/DotTraceIP.git
cd DotTraceIP
pip install -r requirements.txt
```

## Использование

Запустите главный файл для перехода в интерактивное меню терминала:

\`\`\`bash
python main.py
\`\`\`

При первом запуске программа автоматически создаст файлы `target_ips.txt` и `proxies.txt` в корневой директории. Управление пулами потоков, чтением файлов и типами прокси-серверов осуществляется через внутренние настройки приложения.
